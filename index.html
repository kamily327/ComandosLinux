Comandos Linux Essenciais

Categoria Comando Descrição
Arquivos & Navegação ls Listar arquivos e pastas
ls -la Listar tudo (inclusive ocultos)
cd nome_da_pasta Entrar numa pasta
pwd Mostrar onde você está
mkdir nome Criar pasta
rm -r nome Apagar pasta e conteúdo
rm arquivo Apagar arquivo
cp origem destino Copiar arquivo
mv origem destino Mover ou renomear arquivos
touch arquivo.txt Criar arquivo vazio
cat arquivo.txt Mostrar conteúdo do arquivo
Pacotes sudo apt update Atualizar listas de pacotes
sudo apt upgrade Atualizar programas
sudo apt remove nome Remover programa
sudo apt install nome Instalar programa
Processos ps Listar processos
top Monitorar processos em tempo real.
kill PID Encerrar processo (PID aparece no ps ou top).
killall nome_do_programa Encerrar processo pelo nome.
Permissões chmod 755 arquivo Dono pode tudo,grupo e outros podem ler/executar
chmod 777 arquivo Libera tudo pra todo mundo (cuidado)
chown user:group arquivo Mudar dono e grupo do arquivo
Sistema df -h Mostrar espaço em disco
free -h Mostrar uso de memória
uname -a Mostrar infos do kernel
whoami Mostra nome do Usuário logado
ifconfig Mostrar IP e detalhes da rede
